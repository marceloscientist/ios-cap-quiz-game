
# QuizGame 🎯

Aplicativo iOS desenvolvido em **SwiftUI** como projeto prático do treinamento **Capgemini / Proway**.

## 📱 Funcionalidades
- Seleção de tema do quiz
- Fluxo de perguntas com múltipla escolha
- Feedback imediato de resposta correta/incorreta
- Resultado final do quiz
- Arquitetura organizada por features (MVVM)

## 🧱 Arquitetura
- **SwiftUI**
- **MVVM**
- Organização por camadas:
  - App (bootstrap e navegação raiz)
  - Features (/Quiz)
  - Domain / Data / Presentation
  - Core/UI (componentes reutilizáveis)

## 🚀 Tecnologias
- Swift
- SwiftUI
- Combine
